我没有很明确的职业路线，甚至说不指望自己会有职业，也没有特别值得展示的人生规划。这里放的，大多只是某个时间点还没有彻底坏掉的东西；如果它们能在以后被谁用上，当然很好。如果不能，至少曾经被好好放过，不保证以后还有。
