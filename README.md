*Peace in time we never had it so good.*

你好，我是阿绫。

我不太会写程序，也没有打算把自己介绍成程序员。之所以还在用 GitHub，主要是因为确实从开源社区拿到过很多东西：代码、工具、答案，还有一些本来不会遇见的人。所以偶尔也把自己做的东西放回来一点，算不上回报，顶多是不要只拿不放。

做过风俗，唱过歌，现在在做一人字幕组。也写一点字，改一点网页，给喜欢的东西补字幕、整理字体，或者把一些差点散掉的东西暂时放在一起。

我没有很明确的职业路线，也没有特别值得展示的人生规划。这里放的，大多只是某个时间点还没有彻底坏掉的东西；如果它们能在以后被谁用上，当然很好。如果不能，至少曾经被好好放过。
